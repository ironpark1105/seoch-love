# seoch-love

url:https://ironpark1105.github.io/seoch-love/

서초동물사랑센터는 유기견 분양 사이트로 
버려진 강아지를 분양하고 보호하는 사이트입니다.

서초동물사랑센터의 웹페이지는 레이아웃이 복잡하고, 심미적인 측면이
매우 취약하였습니다. 이러한 단점을 보안하여 레이아웃을 간결하고 깔끔하게 
제작하였고 웹페이지 전반의 디자인작업을 진행했습니다.
